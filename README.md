📝 Task Tracker Application

A full-stack Task Management System that allows users to create, manage, and track their daily tasks efficiently.

🚀 Features

* ➕ Create new tasks
* 📋 View all tasks
* ✏️ Update task details
* ❌ Delete tasks
* 📅 Track task status (Pending / Completed)
* 🔐 REST API-based architecture

🛠️ Tech Stack

* Backend: Spring Boot
* Language: Java
* Database: MySQL
* Build Tool: Maven
* API Testing: Postman
